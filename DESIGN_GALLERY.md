# 🎨 Awesome DESIGN.md 갤러리

안티그래비티 프로젝트에서 즉시 활용 가능한 60여 개의 디자인 시스템 카탈로그입니다.

## 🚀 사용 방법

1. 아래 목록에서 원하는 디자인 서비스의 **[미리보기]** 링크를 클릭하여 디자인을 확인합니다.
2. 마음에 드는 디자인의 이름을 확인한 후 터미널에서 다음 명령을 실행합니다.
   ```powershell
   # 예: Vercel 디자인 적용
   gsd-design apply vercel
   ```
3. 에이전트에게 "적용된 `DESIGN.md`를 참고해서 UI를 구현해줘"라고 요청하세요.

### 🛡️ 백업 기능 안내
`gsd-design apply` 명령은 프로젝트의 기존 `DESIGN.md`를 덮어쓰기 전, 자동으로 `DESIGN.md.bak` 파일을 생성하여 안전하게 보관합니다.
